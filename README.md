❤️ Heart Disease Prediction SystemThis is an end-to-end Machine Learning web application designed to predict the risk of heart disease in patients based on clinical parameters.🚀
Project OverviewThe core of this project is a Classification Model (Decision Tree/Logistic Regression) trained on heart health data. It allows users to input medical metrics through an interactive web interface and receive an instant risk assessment.📊 Patient Metrics Required 

The model analyzes the following 9 features to make a prediction 
1)Age: Patient's age in years.
2)Sex: Biological sex (Male/Female).
3)Chest Pain Type: Categorical scale from 0 to 3.
4)Resting Blood Pressure: Measured in mm Hg.
5)Cholesterol Level: Serum cholesterol in mg/dl.
6)Fasting Blood Sugar: Whether the level is $> 120$ mg/dl.
7)Resting ECG: Results categorized from 0 to 2.
8)Max Heart Rate: Maximum heart rate achieved during exercise.
9)Exercise Induced Angina: Presence of chest pain during exercise (Yes/No).
